# Política de Privacidad - UTP Comunidades

**Última actualización:** Abril 2026

---

## 1. Información General

UTP Comunidades ("nosotros", "nuestra", "la aplicación") se compromete a proteger la privacidad de sus usuarios. Esta política de privacidad explica qué datos recolectamos, cómo los usamos y tus derechos conforme al Reglamento General de Protección de Datos (GDPR) y otras leyes de privacidad aplicables.

---

## 2. Datos que Recoleccionamos

### 2.1 Datos Personales
- **Correo electrónico:** (@utp.edu.pe) - Obligatorio para registro y autenticación
- **Nombre:** - Obligatorio para identificación en la plataforma
- **Apellido:** - Opcional
- **Carrera:** - Opcional, para mostrar en perfil
- **Ciclo:** - Opcional, para mostrar en perfil
- **Biografía:** - Opcional, descripción personal
- **Gustos/Intereses:** - Opcional, lista de intereses personales
- **Foto de perfil:** - Opcional, imagen subida por el usuario

### 2.2 Datos de Uso
- **Publicaciones:** Contenido que creas en comunidades
- **Comentarios:** Comentarios en publicaciones
- **Likes:** Reacciones a publicaciones
- **Membresías:** Comunidades a las que te unes
- **Seguidores:** Relaciones de seguimiento entre usuarios
- **Notificaciones:** Preferencias de notificaciones

### 2.3 Datos Técnicos
- **Dirección IP:** Para seguridad y análisis
- **Dispositivo:** Tipo de dispositivo y sistema operativo
- **User Agent:** Información del navegador/app
- **Logs de actividad:** Acciones realizadas en la plataforma

## 3. Cómo Usamos tus Datos

### 3.1 Para el Funcionamiento de la Aplicación
- Autenticación y gestión de sesión
- Creación y gestión de tu perfil
- Publicación de contenido en comunidades
- Interacción con otros usuarios (likes, comentarios)
- Gestión de membresías en comunidades

### 3.2 Para Seguridad
- Prevenir fraudes y abuso
- Detectar actividades sospechosas
- Proteger la integridad de la plataforma
- Cumplir con obligaciones legales

### 3.3 Para Mejorar el Servicio
- Análisis de uso de la aplicación
- Identificación de áreas de mejora
- Desarrollo de nuevas funcionalidades

---

## 4. Con Quién Compartimos tus Datos

### 4.1 Proveedores de Servicios
- **Firebase (Google):** Para autenticación y almacenamiento
- **Railway:** Para hosting de la base de datos y API
- **Neon (PostgreSQL):** Para almacenamiento de datos

### 4.2 No Vendemos tus Datos
- Nunca vendemos tus datos personales a terceros
- No compartimos tus datos con fines de marketing

### 4.3 Divulgación Legal
- Podemos divulgar datos si es requerido por ley
- Para proteger nuestros derechos legales
- Para prevenir fraudes o delitos

---

## 5. Seguridad de tus Datos

### 5.1 Medidas de Seguridad
- **Encriptación:** TLS/SSL para todas las comunicaciones
- **Autenticación:** Firebase Auth con verificación de email
- **Tokens:** JWT con expiración para sesiones
- **Sanitización:** Input sanitizado para prevenir XSS
- **Rate Limiting:** Protección contra ataques DoS
- **Validación:** Validación de todos los inputs

### 5.2 Retención de Datos
- **Datos de cuenta:** Mientras la cuenta esté activa
- **Contenido:** Mientras la cuenta esté activa
- **Logs:** Máximo 90 días
- **Datos eliminados:** Borrados permanentemente al eliminar cuenta

---

## 6. Tus Derechos (GDPR)

### 6.1 Derecho de Acceso
Puedes solicitar una copia de todos tus datos personales que tenemos.

### 6.2 Derecho de Rectificación
Puedes corregir datos inexactos o incompletos en tu perfil.

### 6.3 Derecho de Eliminación (Derecho al Olvido)
Puedes solicitar la eliminación completa de tu cuenta y todos tus datos. Esto incluye:
- Todas tus publicaciones
- Todos tus comentarios
- Todos tus likes
- Todas tus membresías
- Todos tus datos personales

**Para eliminar tu cuenta:**
1. Ve a Configuración en la app
2. Selecciona "Eliminar cuenta"
3. Confirma la eliminación
4. Todos tus datos serán borrados permanentemente

### 6.4 Derecho de Portabilidad
Puedes solicitar una copia de tus datos en un formato estructurado.

### 6.5 Derecho de Oposición
Puedes oponerte al procesamiento de tus datos para ciertos fines.

### 6.6 Derecho de Restricción
Puedes solicitar que limitemos el procesamiento de tus datos.
